## Why It Works
- Review/testimonial format has 45% higher IPM + 17% better Day 7 retention
- Talking head = parasocial bond (viewer feels like a friend is recommending)
- HeyGen avatars bypass the "I don't want to be on camera" problem
- Can produce 5 variants from different "people" in one batch

## Production Rules
- **Casual. Not scripted-sounding.** Write the script like a text message, not a presentation.
- **Intercut with screen recording.** Avatar talks → show app → avatar reacts. Don't let the avatar talk for more than 5 seconds straight.
- **Choose young, relatable avatars.** 20s. Casual clothing. Home background. Not corporate.
- **Add imperfections.** Start mid-sentence. Include "um" and "like." Perfect delivery = uncanny valley.

## Structure (30-45 seconds)

```
[0-2s] Avatar, mid-thought: "ok so i have to tell you about this alarm app"
[2-8s] Quick personal setup: "i was sleeping through literally everything"
[8-15s] The discovery: "someone told me to try this app where you scan a qr code"
[15-25s] Screen recording: app demo intercut with avatar reactions
[25-35s] The result: "i've woken up on time every day for 2 weeks"
[35-40s] "and it's FREE. like the qr scan thing is actually free. link in bio."
```

**Psychological triggers**: Social proof (someone else tried it) + Authority (they tested it for 2 weeks) + Novelty (QR scan alarm = new concept)

## HeyGen Batch Production

**Weekly workflow**:
1. Write 3 scripts (use variations above as starting points)
2. Open HeyGen → select avatar → paste script → generate
3. Download all 3
4. Open CapCut → intercut screen recordings → add captions
5. Queue in Buffer

**Avatar rotation**: Use 3-4 different avatars to simulate different "reviewers." Don't use the same avatar every time.
