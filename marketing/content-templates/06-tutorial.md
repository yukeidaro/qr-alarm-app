# 06: Speed Tutorial

**Not a polished how-to. A breathless, fast, "let me show you this thing real quick" format.**

> **2026-04-07 update (実視聴反映)**
> - Instagram版は45-50秒、末尾10秒に `00-VALUE-ASSET.md` 必須（チュートリアルだけで終わらせない）
> - チュートリアル本体は20-30秒に圧縮し、残り10秒は社会的証明、最後10秒で価値リスト
> - キャプション: `How to never snooze again / Save it for later & Follow @scanalarm.app`
> - 早口ナレーションは**オリジナル音源**として登録（音源で他者がリミックスしてリーチ拡張）

## Why It Works
- Highest save rate of all formats (people save to reference later)
- Saves = 2x algorithm weight
- Positions as friend sharing a discovery, not brand advertising
- Screen recordings = undeniable proof the app is real and works

## Production Rules
- **Screen record at actual wake-up time.** Not a demo at 2pm.
- **Talk fast.** Like you're telling a friend between classes. Not presenting.
- **No intro.** No "hey guys." Start mid-action.
- **30 seconds MAX.** If it takes longer, you're overexplaining.

## Structure (20-30 seconds)

```
[0-1s] Mid-action start. Already in the app. "ok so"
[1-5s] "you download this — free — set your alarm —"
[5-10s] "pick qr code as the dismiss method —"
[10-15s] "scan any qr code. i use one on my bathroom mirror."
[15-20s] "tomorrow morning: alarm goes off, only way to stop it is walk there and scan."
[20-25s] Show alarm going off → walking → scanning → silence.
[25-30s] "save this. you'll need it."
```

**Psychological triggers**: Urgency (fast pace) + Curiosity (what is this?) + Social currency (sharing makes you look helpful)
