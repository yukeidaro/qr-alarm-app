## Day 1 — 4/2 (木): Apple Developer Program 支払い

### やること
Apple Developer Program の年会費を支払い、アカウントを有効化する。

### 手順

1. https://developer.apple.com/account/ にログイン
2. 上部に「Enrollment」や「Complete Your Purchase」の案内が出ているはず → クリック
3. 年会費 **12,980円/年**（$99 USD）をクレジットカードで支払い
4. 支払い後、承認メールが届くまで待つ
   - 個人アカウント: 通常 **24〜48時間**
   - 日本の場合: 3〜5営業日かかることも
5. 承認されたら https://developer.apple.com/account/ で「Certificates, IDs & Profiles」が見えることを確認

### 注意事項
- Apple IDの名前が本名と一致していないと遅延する
- 2FAが有効になっていること（必須）
- 承認待ちの間に他のタスクを進めてOK

### 確認ポイント
- [ ] 年会費支払い完了
- [ ] 承認メール受信（数日後）
- [ ] developer.apple.com で証明書メニューが見える

## Day 3 — 4/4 (土): AdMobアカウント作成 + 広告ユニットID取得

### やること
Google AdMobアカウントを作成し、6つの広告ユニットIDを取得する。

### 手順

1. **AdMobにサインアップ**
   - https://admob.google.com/ にアクセス
   - Googleアカウントでログイン → 「始める」
   - 支払い情報を入力（住所、電話番号）
   - 利用規約に同意

2. **iOSアプリを登録**
   - 左メニュー「アプリ」→「アプリを追加」
   - プラットフォーム: **iOS**
   - 「アプリストアで公開されていません」を選択
   - アプリ名: **ScanAlarm**

3. **Androidアプリを登録**
   - 同様に「アプリを追加」→ **Android** → **ScanAlarm**

4. **広告ユニットを作成（iOS — 3つ）**

   | 広告ユニット名 | フォーマット | 用途 |
   |--------------|------------|------|
   | `ringing-banner-ios` | バナー | Ringing画面 |
   | `snooze-banner-ios` | バナー | Snooze画面 |
   | `rewarded-snooze-ios` | リワード | 3回目スヌーズ |

5. **広告ユニットを作成（Android — 3つ）**
   - `ringing-banner-android`, `snooze-banner-android`, `rewarded-snooze-android`

6. **IDをメモして Copilot に伝える**
   - 広告ユニットID（`ca-app-pub-XXXX/YYYY` 形式）× 6個
   - アプリID（`ca-app-pub-XXXX~YYYY` 形式）× 2個（iOS, Android）
   - → Copilotが `services/adService.ts` と `app.json` を自動更新します

### 確認ポイント
- [ ] AdMobアカウント作成完了
- [ ] iOS用: アプリ登録 + 3広告ユニット作成
- [ ] Android用: アプリ登録 + 3広告ユニット作成
- [ ] 8個のIDをメモ

## Day 4 — 4/5 (日): スクリーンショット撮影 + メタデータ準備

### やること
App Store / Google Play に必要なスクリーンショットとアプリ説明文を準備する。

### 必要なスクリーンショット

| デバイスサイズ | 解像度 | 必要枚数 |
|-------------|--------|---------|
| iPhone 6.7" (15 Pro Max) | 1290 x 2796 | 3〜5枚 |
| iPhone 6.5" (11 Pro Max) | 1242 x 2688 | 3〜5枚 |
| iPhone 5.5" (8 Plus) | 1242 x 2208 | 3〜5枚 |

### 撮影する画面（推奨5枚）
1. **ホーム画面** — アラーム一覧
2. **アラーム編集** — 時間設定 + 曜日選択
3. **鳴動画面** — アラームが鳴っている状態
4. **QRスキャン** — カメラでQRコードをスキャン中
5. **スヌーズ画面** — カウントダウン表示

### 撮影方法（実機推奨）
- **iPhone**: 電源 + 音量上 を同時押し
- **Android**: 電源 + 音量下 を同時押し
- Expo Goで動かして撮影すればOK

### メタデータ（コピペ用）

**アプリ名**: ScanAlarm

**サブタイトル**:
- JA: QRコードで起きるアラーム
- EN: QR Code Alarm Clock

**キーワード** (カンマ区切り):
- JA: アラーム,目覚まし,QRコード,バーコード,起きる,朝,スヌーズ,スキャン
- EN: alarm,wake,QR,barcode,scan,morning,snooze,clock

**説明文**:
> ScanAlarmは、QRコードやバーコードをスキャンしないと止められない目覚ましアプリです。
> ベッドから出ないとアラームが止まらないので、二度寝を防止できます。
>
> 主な機能:
> - QRコード/バーコードスキャンでアラーム解除
> - スヌーズ機能（回数制限付き）
> - 複数アラームの設定
> - 曜日ごとの繰り返し設定
> - カスタムアラーム音
> - 日本語・英語・韓国語・スペイン語対応

**カテゴリ**: ユーティリティ (Utilities)

- [ ] アプリアイコン最終確認 (1024x1024 PNG)
