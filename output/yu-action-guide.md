## Day 1 — 4/2 (木): Apple Developer Program 支払い

### やること
Apple Developer Program の年会費を支払い、アカウントを有効化する。

### 手順

1. https://developer.apple.com/account/ にログイン
2. 上部に「Enrollment」や「Complete Your Purchase」の案内が出ているはず → クリック
3. 年会費 **12,980円/年**（$99 USD）をクレジットカードで支払い
4. 支払い後、承認メールが届くまで待つ
   - 個人アカウント: 通常 **24〜48時間**
   - 日本の場合: 3〜5営業日かかることも
5. 承認されたら https://developer.apple.com/account/ で「Certificates, IDs & Profiles」が見えることを確認

### 注意事項
- Apple IDの名前が本名と一致していないと遅延する
- 2FAが有効になっていること（必須）
- 承認待ちの間に他のタスクを進めてOK

### 確認ポイント
- [x] 年会費支払い完了
- [ ] 承認メール受信（数日後）

## Day 2 — 4/3 (金): Google Play Console 登録 + 本人確認

### アカウント登録（完了済み）
- [x] $25支払い完了（4/2）
- [x] Play Consoleダッシュボードにアクセス可能

### 本人確認（未完了 — プロダクション公開に必須）

本人確認を完了しないとプロダクション公開がブロックされる。
内部テストは本人確認なしでも可能なので、並行して進められる。

#### 手順

1. https://play.google.com/console/ にログイン
2. 左サイドバー → **Setup** → **Developer account** → **Identity verification**
   - ダッシュボード上部にバナー警告が出ている場合はそこからも進める
3. **「Verify identity」** ボタンをクリック
4. Persona（Google提携の認証サービス）のフローが開始される:

   **Step 1: 身分証の種類を選択**
   - Passport / Driver's License / National ID Card から選ぶ
   - **パスポートが最も確実**（英語表記があり認識精度が高い）
   - 運転免許証、マイナンバーカード（表面のみ）も可

   **Step 2: 身分証を撮影**
   - PCだとQRコードが表示される → **スマホで撮影する方が成功率が高い**
   - 四隅が切れない、反射しない、ぼやけない写真を撮る
   - 明るい場所で、証明書を平らに置いて撮影

   **Step 3: セルフィー撮影（ライブ）**
   - リアルタイムで顔を撮影する（静止画ではない）
   - メガネ、帽子、マスクは外す
   - 身分証の顔写真と照合される

5. 送信して完了 → Googleからメールで結果通知

#### 承認までの時間
- 通常 **1〜3営業日**（早ければ数時間）
- 書類不鮮明で差し戻しの場合、追加で数日

#### よくある失敗パターン
- 身分証の写真が不鮮明（反射・ぼやけ・四隅切れ）→ スマホカメラ推奨
- Play Consoleの登録名と身分証の名前が不一致 → **Yu Asano** で統一されていればOK
- マイナンバーカードの裏面を送ってしまう → **表面のみ**（番号が写る裏面は不可）
- ブラウザのカメラ権限がブロックされている → 設定で許可する

## Day 4 — 4/5 (日): スクリーンショット撮影 + メタデータ準備

### やること
App Store / Google Play に必要なスクリーンショットとアプリ説明文を準備する。

### 必要なスクリーンショット

| デバイスサイズ | 解像度 | 必要枚数 |
|-------------|--------|---------|
| iPhone 6.7" (15 Pro Max) | 1290 x 2796 | 3〜5枚 |
| iPhone 6.5" (11 Pro Max) | 1242 x 2688 | 3〜5枚 |
| iPhone 5.5" (8 Plus) | 1242 x 2208 | 3〜5枚 |

### 撮影する画面（推奨5枚）
1. **ホーム画面** — アラーム一覧
2. **アラーム編集** — 時間設定 + 曜日選択
3. **鳴動画面** — アラームが鳴っている状態
4. **QRスキャン** — カメラでQRコードをスキャン中
5. **スヌーズ画面** — カウントダウン表示

### 撮影方法（実機推奨）
- **iPhone**: 電源 + 音量上 を同時押し
- **Android**: 電源 + 音量下 を同時押し
- Expo Goで動かして撮影すればOK

### メタデータ（コピペ用）

**アプリ名**: ScanAlarm

**サブタイトル**:
- JA: QRコードで起きるアラーム
- EN: QR Code Alarm Clock

**キーワード** (カンマ区切り):
- JA: アラーム,目覚まし,QRコード,バーコード,起きる,朝,スヌーズ,スキャン
- EN: alarm,wake,QR,barcode,scan,morning,snooze,clock

**説明文**:
> ScanAlarmは、QRコードやバーコードをスキャンしないと止められない目覚ましアプリです。
> ベッドから出ないとアラームが止まらないので、二度寝を防止できます。
>
> 主な機能:
> - QRコード/バーコードスキャンでアラーム解除
> - スヌーズ機能（回数制限付き）
> - 複数アラームの設定
> - 曜日ごとの繰り返し設定
> - カスタムアラーム音
> - 日本語・英語・韓国語・スペイン語対応

**カテゴリ**: ユーティリティ (Utilities)
