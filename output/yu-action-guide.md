## 1. AdMobアカウント作成 + Ad Unit ID取得
**期限**: 4/3 | **所要時間**: 約30分 | **優先度**: 高

### 手順

1. **AdMobにサインアップ**
   - https://admob.google.com/ にアクセス
   - Googleアカウントでログイン
   - 「始める」をクリック
   - 支払い情報を入力（住所、電話番号）
   - 利用規約に同意

2. **iOSアプリを登録**
   - 左メニュー「アプリ」→「アプリを追加」
   - プラットフォーム: **iOS**
   - 「アプリストアで公開されていません」を選択
   - アプリ名: **ScanAlarm**
   - 追加

3. **Androidアプリを登録**
   - 同様に「アプリを追加」→ **Android** → **ScanAlarm**

4. **広告ユニットを作成（iOS）**
   - アプリ一覧からiOS ScanAlarmを選択
   - 「広告ユニット」→「広告ユニットを追加」
   - 以下3つを作成:

   | 広告ユニット名 | フォーマット | 用途 |
   |--------------|------------|------|
   | `ringing-banner-ios` | バナー | Ringing画面 上部 + ボタン上 |
   | `snooze-banner-ios` | バナー | Snooze画面 上部 + ボタン上 |
   | `rewarded-snooze-ios` | リワード | 3回目スヌーズ時のリワード広告 |

5. **広告ユニットを作成（Android）**
   - 同様に3つ作成:
   - `ringing-banner-android`, `snooze-banner-android`, `rewarded-snooze-android`

6. **IDをメモ**
   - 各広告ユニットの ID（`ca-app-pub-XXXX/YYYY` 形式）をメモ
   - アプリID（`ca-app-pub-XXXX~YYYY` 形式）もメモ

7. **コードに反映**（これはCopilotに依頼可能）
   - メモしたIDを伝えてください
   - `services/adService.ts` と `app.json` を更新します

### 確認ポイント
- [ ] AdMobアカウント作成完了
- [ ] iOS用アプリ登録
- [ ] Android用アプリ登録
- [ ] 6つの広告ユニットID取得
- [ ] 2つのアプリID取得

| iPhone 6.5" (11 Pro Max) | 1242 x 2688 | 3-5枚 |
| iPhone 5.5" (8 Plus) | 1242 x 2208 | 3-5枚 |

### 撮影する画面（推奨5枚）
1. **ホーム画面** — アラーム一覧が表示されている状態
2. **アラーム編集** — 時間設定 + 曜日選択
3. **鳴動画面** — アラームが鳴っている状態
4. **QRスキャン** — カメラでQRコードをスキャン中
5. **スヌーズ画面** — カウントダウン表示

### 撮影方法
```bash

# Xcodeを起動 → シミュレーター選択 → Cmd+S でスクリーンショット

# または実機で撮影
# iPhone → 電源 + 音量上 を同時押し
```

### メタデータ（日本語 + 英語）

**アプリ名**: ScanAlarm (30文字以内)

**サブタイトル** (30文字以内):
- JA: QRコードで起きるアラーム
- EN: QR Code Alarm Clock

**キーワード** (100文字以内、カンマ区切り):
- JA: アラーム,目覚まし,QRコード,バーコード,起きる,朝,スヌーズ,スキャン
- EN: alarm,wake,QR,barcode,scan,morning,snooze,clock

**説明文** (4000文字以内):
> JA:
> ScanAlarmは、QRコードやバーコードをスキャンしないと止められない目覚ましアプリです。
>
> ベッドから出ないとアラームが止まらないので、二度寝を防止できます。
>
> 主な機能:
> - QRコード/バーコードスキャンでアラーム解除
> - スヌーズ機能（回数制限付き）
> - 複数アラームの設定
> - 曜日ごとの繰り返し設定
> - カスタムアラーム音
> - 日本語・英語・韓国語・スペイン語対応

**カテゴリ**: ユーティリティ (Utilities)

**年齢制限**: 4+ (広告あり → 実際は要確認)

## 3. Google Play Console 登録
**期限**: 4/5 | **所要時間**: 約20分（承認に数日かかる場合あり） | **優先度**: 高

### 手順

1. **デベロッパーアカウント登録**
   - https://play.google.com/apps/publish/signup/ にアクセス
   - Googleアカウントでログイン
   - 登録料 **$25** を支払い（一回のみ）

2. **本人確認**（2025年以降必須）
   - 有効な身分証明書（パスポートまたは運転免許証）
   - 本名と一致するクレジットカード
   - 承認まで: 通常数日、最大2週間

3. **アプリリスティング作成**
   - Play Console → 「アプリを作成」
   - アプリ名: **ScanAlarm**
   - デフォルト言語: **日本語**
   - アプリのタイプ: **アプリ** (ゲームではない)
   - 無料/有料: **無料**
   - 宣言事項にチェック

4. **ストア掲載情報**（スクリーンショット等はTask 2と共通）
   - 簡単な説明 (80文字): QRコードをスキャンして止める目覚ましアプリ
   - 詳しい説明: App Store版と同様
   - スクリーンショット: 最低2枚、推奨5枚
   - フィーチャーグラフィック: 1024 x 500 px（任意だが推奨）

5. **コンテンツのレーティング**
   - アンケートに回答（暴力なし、広告あり等）
   - IARCレーティング自動取得

6. **プライバシーポリシーURL設定**
   - Privacy policyはこちらで自動作成済み
   - GitHub Pagesまたは任意のホスティングにデプロイ後URLを設定

### 注意事項
- 承認に時間がかかるため、**早めの登録を推奨**
- 初回リリースは「内部テスト」トラックから始める（20人まで、審査不要）

### 確認ポイント
- [ ] $25支払い完了
- [ ] 本人確認完了
- [ ] アプリリスティング作成
